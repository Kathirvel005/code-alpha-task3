# Task Automation with Python Scripts

## Objective
Automate a small real-life task using Python.

## Selected Task
Extract all email addresses from a text file and save them to another file.

## Technologies Used
- Python
- Regular Expressions (re)
- File Handling

## Files
- email_extractor.py
- input.txt
- emails.txt (generated after execution)

## How It Works
1. Reads text from `input.txt`.
2. Uses a regular expression to find email addresses.
3. Saves all extracted emails into `emails.txt`.
4. Displays the number of emails found.

## Sample Output

Successfully extracted 3 email(s).
Saved to 'emails.txt'.
